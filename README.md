# blahblah.tech

## Prerequisite

### Install nvm, Node.js

```
$ url -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.6/install.sh | bash
$ command -v nvm
$ nvm install 8.9.0
$ nvm alias default 8.9.0
$ node --version
v8.9.0
```

### Install Yarn

```
$ npm install -g yarn
$ yarn --version
```

## Get Started

### Install project

```
$ nvm use
$ yarn
```

### Build

```
$ yarn build
```

### Run

```
$ yarn start
```
